# Get Checkout Link

Returns the checkout link for a plan. This endpoint will only work for paid Plans.

## Code Sample

```typescript
import { Salable } from '@salable/node-sdk';

const salable = new Salable('{{API_KEY}}');

const checkoutLink = await api.plans.getCheckoutLink('{{PLAN_UUID}}', {
  cancelUrl: 'https://example.com/cancel',
  successUrl: 'https://example.com/success',
  granteeId: 'userId-1234',
  member: 'member-123',
});
```

## Parameters

##### planId (_required_)

_Type:_ `string`

The `uuid` of the Plan to return the Currencies from
